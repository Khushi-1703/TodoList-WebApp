# 📝 SlayList — Todo List App

A clean and responsive Todo List web app built to help you stay organized, productive, and on top of your daily tasks.
TaskFlow lets users add, manage, complete, and delete tasks with a smooth and modern user experience.

---

## 🚀 Features

* ➕ Add new tasks instantly
* ✅ Mark tasks as completed
* 🗑️ Delete tasks easily
* ✏️ Edit existing tasks
* 📋 Organized and minimal UI
* 📱 Fully responsive design
* 🎨 Smooth and modern user interface
* 📌 Scrollable task list for better usability

---

## 🛠️ Tech Stack

* **React.js** – Frontend library
* **Tailwind CSS** – Styling
* **JavaScript (ES6+)** – Functionality
* **Vite** – Fast development build tool

## ⚙️ Installation & Setup

Follow these steps to run the project locally:

### 1. Clone the repository

```bash
git clone https://github.com/your-username/todo-list-app.git
```

### 2. Navigate into the project folder

```bash
cd todo-list-app
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the development server

```bash
npm run dev
```

### 5. Open in browser

```bash
http://localhost:5173
```

## 🎯 How It Works

1. Type a task into the input field
2. Click the **Add** button
3. Your task appears in the list
4. Mark it complete when done
5. Edit or delete tasks anytime

Simple, fast, and effective ✨
